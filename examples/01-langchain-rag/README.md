# LangChain RAG + CtxVault

Minimal RAG example using:

- LangChain
- CtxVault API (local semantic store)
- Optional OpenAI LLM

Shows how to use CtxVault as a drop-in retriever for LangChain.

---

## What this shows

- semantic search via CtxVault
- automatic indexing of local docs
- works with or without an LLM
- plug-and-play API integration

---

## Quick start

```bash
pip install -r requirements.txt
python langchain_rag_example.py
```

Then ask questions in the terminal.

Works even **without OPENAI_API_KEY** (retrieval-only mode).

---

## With OpenAI (optional)

```bash
export OPENAI_API_KEY=your_key
python langchain_rag_example.py
```

Now answers are generated by the LLM using retrieved context.
